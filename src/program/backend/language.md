## 一、如何快速入门

一门编程语言的入门学习最好参考网上优质网站教程、视频教程和参考书籍，尤其对于有编程基础的人而言，通过网站教程和参考书是最快的入门方式，虽然没有视频教程来得直观，但是学完后印象会比较深刻。

::: tip 

学习一门语言的基本顺序是:

- 了解数据结构
- 了解变量声明与赋值
- 了解判断和循环体结构
- 了解函数声明
- 了解输入输出

:::
